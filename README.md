# 🤖 HybridRAG Chatbot

HybridRAG Chatbot is an intelligent AI assistant that combines **PDF-based Retrieval-Augmented Generation (RAG)** with **live internet search** to deliver accurate and context-aware answers. It uses a local LLM (TinyLlama via Ollama), ensuring privacy, efficiency, and zero dependency on paid APIs.

---

## 🚀 Features

- 📄 **PDF Knowledge Base** – Ask questions from your own documents  
- 🌐 **Internet Search Fallback** – Fetch answers when PDFs don’t contain relevant info  
- 🧠 **RAG Architecture** – Combines retrieval + generation for accurate responses  
- 🤖 **Local LLM (Ollama)** – No OpenAI API required  
- ⚡ **Fast Vector Search (FAISS)**  
- 💬 **Interactive Chat UI** (built with Flask)

---

## 🏗️ Tech Stack

- **Backend:** Flask  
- **AI Framework:** LangChain  
- **Embeddings:** HuggingFace (MiniLM)  
- **Vector DB:** FAISS  
- **LLM:** TinyLlama (via Ollama)  
- **Search Engine:** DuckDuckGo
